    ROW( of: ROWS, cells: tab) :ROW
    Place to store one record (and stats on how it is used; e.g. i.evaled=true if we touch the y values.

    local ROW = obj("ROW", function(i,of,cells) 
  i._of,i.cells,i.evaled = of,cells,false end)
